# Cow Face 10-Second Sequence Dataset

This folder was generated by `yolo_cow_face/create_10s_face_sequences.py`.

## Data Science Rules Applied

- Video selection is reproducible with seed `42`.
- Cow `409` is excluded by default because it is not classified in the current health list.
- Videos without cow IDs or without verified healthy/unhealthy labels are excluded from the clean sequence dataset.
- Selection uses round-robin sampling by cow so all classified cows are covered before adding additional videos.
- Each completed sequence is exactly `10` seconds represented as `240` frames at `24` FPS.
- When more than one face is detected, the largest detected face crop is used as the nearest cow.
- Per-sequence metadata and per-frame detection/fill status are saved for auditability.

## Important Files

- `candidate_inventory.csv`: all eligible candidate videos after filtering.
- `candidate_selected_by_cow.csv` / `.md`: candidate and selected counts per cow.
- `selection_manifest.csv`: selected videos and labels.
- `selected_videos_table.md`: exact selected videos in presentation-friendly table form.
- `completed_manifest.csv`: completed sequences after crop extraction.
- `processing_statistics.json`: full processing summary.
- `sequences/*/metadata.json`: metadata for each sequence.
- `sequences/*/frames.csv`: per-frame source frame, detection status, confidence, and box.
