# V2 folder — protocol, artifacts, and Rorqual results

## Purpose

Bundle for the **V2 Holstein transfer protocol**: stratified cow-level CV with **two validation cows per fold** (14 folds), **four held-out test cows fixed to V1** (`363`, `403`, `404`, `408`), optional denser temporal sampling, sliding-window / MC eval flags, and pooled diagnostics (Brier, NLL, ECE).

## Relation to V1

| Setting | V1 (example vast run) | V2 (this bundle) |
|--------|------------------------|------------------|
| Val cows per fold | 4 → 7 folds | 2 → **14 folds** |
| Test cows | 4 (balanced selection from seed) | **Same IDs as V1** (`--test-cow-ids`) |
| Default output dir names | e.g. `holstein_task1_dann_vast_run` | `holstein_task1_dann_v2_run`, `holstein_task1_weak_gce_v2_run` |

Training cost roughly **doubles** vs 7-fold CV because the number of inner folds doubles.
