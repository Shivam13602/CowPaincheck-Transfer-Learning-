# V1 experiment archive — Task1-only DANN + weak-label GCE (Vast.ai)

This folder is a **frozen snapshot** of the May 4, 2026 run on a Vast.ai GPU instance: **Holstein cow-held-out** adaptation with **UCAPS v2.9** pretrained weights, followed by **weak-proxy fine-tuning** with **GCE** loss, plus a **reference UCAPS held-out evaluation** of the pretrained task1 checkpoints.

---

## 1. What was run (pipeline)

| Stage | Script | Purpose |
| --- | --- | --- |
| Dry run | `dann_adapt_v2.9.py` `--dry-run` | Validate manifest, splits, paths (no training). |
| Task1-only DANN | `dann_adapt_v2.9.py` | Domain-adapt UCAPS→Holstein; **source Task2 loss = 0**, **target weak BCE = 0**; init from `best_model_v2.9_task1_fold_0.pt`. |
| Weak-label GCE | `weak_label_adapt_v2.9.py` | Fine-tune on Holstein `video_health_status` with **task1 loss = GCE**, `gce_q=0.7`; init from **task2** fold 0 UCAPS checkpoint. |
| UCAPS eval (reference) | `evaluate_test_set_v2.9_cli.py` | Held-out **animals 14 and 17**, **task1**, **mean_logits** ensemble over folds 0–8 (not Holstein). |

Orchestration on the server matched `run_task1_vast.sh`, with output directories renamed for this run:

- DANN: `holstein_task1_dann_vast_run` (and `holstein_task1_dann_vast_run_dry_run` for the dry run).
- Weak GCE: `holstein_task1_weak_gce_vast_run`.

## 3. Incidents and fixes (important for reproducibility)

1. **Missing file on first weak-label attempt:** `weak_label_adapt_v2.9.py` imports `evaluate_test_set_v2.9_cli.py` via `ucaps_v29_eval_loader.py`. That file was **not** in the original zip list in `package_necessary_filesV1.ps1`. **Fix:** add `evaluate_test_set_v2.9_cli.py` to the repo `Dann transfer/` folder and to the packaging script; upload to the instance.

2. **PyTorch 2.6 `torch.load` default (`weights_only=True`):** After all seven weak-label folds trained, **test ensemble** crashed when loading `best_weak_task1.pt` (`pickle.UnpicklingError` / `PosixPath` not allowlisted). **Fix:** all `torch.load(...)` in `weak_label_adapt_v2.9.py` now pass **`weights_only=False`** (trusted checkpoints). A copy of the patched file is in this folder as `weak_label_adapt_v2.9_patched_weights_only.py`.

3. **First full weak run** completed folds but failed at ensemble; **second run** (patched script) completed end-to-end. Logs: `logs/v1_finish.log` (failed tail), `logs/weak_only_retry.log` (success).
